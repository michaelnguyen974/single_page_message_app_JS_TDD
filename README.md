# single_page_message_app_JS_TDD
## User stories
```
As a user, 
I want to see a title. 

As a user, 
I want to see a textbox and a button. 

As a user,
I want to be able to submit a message.

As a user, 
I want to see all my messages displayed. 

As a user, 
I want to see a timestamp on my messages. 

As a user, 
I want to see my messages in reverse chronological order. 
```
## Client stories
```
As a client, 
so my app is easy to maintain, 
the logic should be encapsulated in classes. 

As a client, 
so my app is reliable, 
the logic should be fully tested, with 95% coverage. 
```
